再谈内联元素

外在盒子是内联盒子的元素称为内联元素

内联盒模型

1. 内容区域(content area)

通常情况下，文字的选中背景可以反映出内容区域的范围(排除垂直padding的情况)。

2. 内联盒子(inline box)

内联盒子不会让内容成块显示，而是排成一行，直到这行空间不够再换行。

  • 内联盒子
  • 匿名内联盒子(光秃秃的文字)

    3. 行框盒子(line box)

    每一行就是一个行框盒子

    4. 包含块(containing box)

    包含块由一个或多个行框盒子组成，这里的包含块就是<p>标签

幽灵空白结点(特别重要)

Each line box starts with a zero-width inline box with the element’s font and line height properties. We call that imaginary box a “strut”.

每个行框盒子的前面有一个空白结点，这个空白结点永远透明，不占据任何宽度，看不见也无法通过脚本获取。
